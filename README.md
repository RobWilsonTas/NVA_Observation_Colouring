The millions of NVA observations present a situation where we want to randomly assign colours to different species, but have that colour stick so it always has the colour it is assigned, for points, lines and polygons

This script converts the text string of the species into a number, and uses that number to assign it a colour

It does this such that every imaginable text string is assigned to one of 30 colours

This is applied through symbology to points, lines or polygons in QGIS
